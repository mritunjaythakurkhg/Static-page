Create Virtual Cards To Make
Online Transactions

Scandapay allows users to create virtual cards. These virtual cards can be used to pay for goods and services online. Users can create a Naira virtual card (NGN Card), or a Dollar virtual card (USD Card).

The experience of the virtual dollar card was also improved by redesigning the confirmation screens, this is to ensure that users confirm a transaction before it goes through.
